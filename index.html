Process

Three simple steps to run your business better.

  • Development Tools
  • The Causality High Performance Platform
  • Migrate selectively & gradually

step 1

Pony is more than just a programming language. It is a collection of ideas that helps you to implement high performance, scalable and reliable applications more reliably and more efficiently. We integrate with widely adopted tools such as Sublime Text, Atom and Visual Studio. Pony programs can be debugged with standard tools like LLDB and GDB.

step 2

The Causality Platform is a generic application framework, based on Pony, that allows you to develop your applications independent of an underlying IT infrastructure to which you want to deploy your applications. Write a program once and run it anywhere, without the overhead of an interpreter or virtual machine. We even support heterogeneous clusters!

step 3

We know you have critical and important applications in-place. They cannot simply be replaced. This is why we have made sure that Causality technology is as interoperable as possible. For example, Pony is compatible with C and C++. It is therefore possible to replace performance critical parts of an application first, and migrate step by step.
